# Arduino RGB Reflex Game (WOKWI)

Simulation of a reflex game using Arduino.

## Components 

- Arduino Uno
- LCD 16x2
- RGB LED
- Resistor 220Ω (4)
- Pushbutton (4)
- Potentiometer

## Features

- Waits a random time between 2 and 5 seconds before starting
- Displays one of four colours: red, green, blue, or yellow
- Yellow is created by combining red and green on the RGB LED
- The player must press the matching button as quickly as possible
- Displays an error message if the wrong button is pressed
- Displays the reaction time on the LCD
- Stores and displays the best reaction time

## Controls

- Press the button matching the displayed colour:
- Red button for red
- Green button for green
- Blue button for blue
- Yellow button for yellow
